Projects

Each tool has its own job, but they all point at the same goal: make chess practice feel concrete and useful.

Play

Play Bots

Jump straight into games against custom bots. The point is not a generic engine match; it is a place to test ideas, get punished for lazy moves, and review what happened after.

  • Different bot profiles with their own style
  • Legal board play with premoves and undo support
  • Game review backed by engine checks
Open Play Bots Source
Study

Bookup

Bring in your games and turn repeated mistakes into a training queue. It is for building a repertoire from positions you actually reach, not from a random opening list.

  • Rapid, blitz, and bullet game imports
  • Training views for positions that need work
  • Smart Theory exports for cleaner study trees
Open Bookup Source
Trace

Chess Connections

Search for a path from one player to another using real public games. Every connection points back to proof games, so the graph is fun without being made up.

  • Rapid, blitz, and bullet wins from Chess.com
  • Readable graph and proof ledger views
  • Cloudflare worker cache for heavier searches
Open Connections Source
